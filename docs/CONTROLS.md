# Controls

All keybindings are configurable via the in-game Settings menu. Custom bindings are saved to `~/.xandaris/keybindings.json`.

## Global

| Action | Default Key |
| --- | --- |
| Pause/Resume | Space |
| Speed: Slow | 1 |
| Speed: Normal | 2 |
| Speed: Fast | 3 |
| Speed: Very Fast | 4 |
| Quick Save | F5 |

## Navigation

| Action | Default Key |
| --- | --- |
| Escape/Back | Escape |
| Open Build Menu | B |
| Open Market | M |
| Open Player Directory | P |
| Focus Home System | Tab |
| Toggle Workforce View | W |

## Menus

| Action | Default Key |
| --- | --- |
| Menu: Up | Up Arrow |
| Menu: Down | Down Arrow |
| Menu: Confirm | Enter |
| Menu: Cancel | Escape |
| Menu: Delete | Backspace |

## Mouse

| Action | Input |
| --- | --- |
| Select entity | Left Click |
| Enter/drill down | Double Click |
| Open build menu for resource | Shift + Left Click |
| Close menu / Cancel construction | Right Click |
| Scroll lists | Mouse Wheel |

## Per-View

### Galaxy View
- Click a system to select it, double-click to enter
- Tab to jump to your home system

### System View
- Click a planet/star to select, double-click to enter planet
- Click a fleet/ship to open the fleet info panel

### Planet View
- B to open build menu
- W to toggle workforce overlay
- Shift+Click a resource to open build menu for it
- Click a shipyard building to open the shipyard UI

### Market View
- Scroll to browse commodities
- Click Buy/Sell buttons to trade

### Fleet Info Panel
- Move, Create Fleet, Join Fleet, Cargo, Disband actions via buttons
- Scroll for fleet/ship details

### Shipyard UI
- Scroll to browse ship types, click to select, Build button to queue

### Construction Queue
- Right-click an item to cancel it

### Load Game Screen
- R to rename a save, D to delete it

### Settings View
- Navigate bindings with Up/Down arrows
- Enter to rebind a key, Escape to cancel rebinding
